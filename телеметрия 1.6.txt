1.0 Запрос идентификации (пишется в модуль памяти при включении)
	0xAD00  	
	-------		
	1. ответное слово
	2. номер прибора	(меняет пользователь)
	3. версия прошивки
	4. версия структуры идентификатора 0х0100
	5. идентификатор 0х0001 (меняет пользователь)
	6. версия телеметрии (1.6)
	7. Кол-во слов строки идентификации
	8.... Строка идентификации (char*2)

	0xAD10 
	------
 	1. Ответное слово

	6. Мертвая зона датчика

	8. Частота излучателя (кГц)
	9. Фильтр
	10. Упаковка
	11. Алгоритм поиска первого вступления (0 - по уровню дискриминации, 1 - ...)	
	12. Длина фрагмента волны начиная с первого вступления (0 - вся волна без обрезания)

	13. КУ (опорный датчик)
	14. Шаг оцифровки
	15. Длина оцифровки
	16. Задержка оцифровки
	17. Мертвая зона датчика
	18. Уровень дискриминации датчика
	19. Частота излучателя (кГц)
	20. Фильтр
	21. Упаковка (0 - нет, 1 - 4:3 по 12 бит, 2 - 2:1 u-Law, 3 - 4:1 ADPCM по 4 бита, 4 - ДКП Низкое, 5 - ДКП Среднее, 6 - ДКП Высокое)

	25. Количество точек на оборот головки в режиме имиджера
	26. Напряжение излучателя (В)
 	27. Ограничение тока двигателя (мА)
	28. Аварийный ток двигателя (мА)

	23. Время опорного датчика минимум по первому вступлению (0.05 мкс)
	------
	Версия 1.2
	------
	24. Состояние датчика Холла (0, 1)
	25. Напряжение излучателя (В)
	26. Напряжение двигателя (В)
	27. Счётчик запросов DSP
	28. Счётчик запросов двигателя
	29. Качество сигнала запроса телеметрии (%)

1.4 Запрос данных цементомера (пишется в модуль памяти)
	0xAD40
	0xoooo  : смещение. если = 0 передача новой пачки данных
	0xssss  : размер. При отсутсвии этих слов - запрос пакета целиком, при запросе несуществующих данных - ответ пустой (ответное слово)
	------
 	1. ответное слово
	2. Время (0.1мс). младшие 2 байта
	3. Время. старшие 2 байта
	4. Время датчика Холла (0.1мс). младшие 2 байта
	5. Время датчика Холла. старшие 2 байта
	6. Счётчик оборотов двигателя (1/6 об)
	7. Счётчик оборотов головки (об)
	8. AX (уе)
	9. AY (уе)
	10. AZ (уе)
	11. AT (short 0.01 гр)
	12. Тип датчика (0 - измерительный датчик, 1 - опорный датчик)
	13. Угол поворота (0.01гр)(ushort)
	14. Амплитуда максимум по всей волне (у.е)
	15. Амплитуда по первому вступлению (у.е)
	16. Время по первому вступлению (0.05 мкс)
	17. КУ
	18. Шаг оцифровки
	19. Длина оцифровки (макс 2028)
	20. Задержка оцифровки  
	21. Упаковка (1 - 4:3 по 12 бит, 2 - 2:1 u-Law, 3 - 4:1 ADPCM по 4 бита)
	22. Размер упакованных данных
	23.... данные волновой картины

2.1 Установить параметры (Версия 1.6) 
	0xAD90     	
	0xpppp: 0x01 - КУ (измерительный датчик)
		0x02 - Шаг оцифровки (0.05мкс)
		0x03 - Длина оцифровки
		0x04 - Задержка оцифровки (0.05мкс)
		0x05 - Мертвая зона датчика (0.05мкс)
		0x06 - Уровень дискриминации датчика
		0x07 - Частота излучателя (кГц) 100...1000
		0x08 - тип фильтра (0-нет,1,2,3)
		0x09 - тип упаковки (0-нет,1,2,3) (1 - 4:3 по 12 бит, 2 - 2:1 u-Law, 3 - 4:1 ADPCM по 4 бита)
		0х0A - Алгоритм поиска первого вступления (0 - по уровню дискриминации, 1 - автоматический)	
		0х0B - Длина фрагмента волны начиная с первого вступления (0 - вся волна без обрезания)

		0x11 - КУ (опорный датчик)
		0x12 - Шаг оцифровки (0.05мкс)
		0x13 - Длина оцифровки
		0x14 - Задержка оцифровки (0.05мкс)
		0x15 - Мертвая зона датчика (0.05мкс)
		0x16 - Уровень дискриминации датчика
                0x17 - Частота излучателя (кГц) 100...1000
		0x18 - тип фильтра (0-нет,1,2,3)
		0x19 - тип упаковки (0-нет,1,2,3) (1 - 4:3 по 12 бит, 2 - 2:1 u-Law, 3 - 4:1 ADPCM по 4 бита)
		0х1A - Алгоритм поиска первого вступления (0 - по уровню дискриминации, 1 - автоматический)	
		0х1B - Длина фрагмента волны начиная с первого вступления (0 - вся волна без обрезания)

		0x30 - Количество волновых картин на оборот головки в режиме цементомера
		0x31 - Количество точек на оборот головки в режиме имиджера

		0x40 - Напряжение излучателя (0..500 В)
      		0x41 - Ограничение тока двигателя (1000..3000 мА)
      		0x42 - Аварийный ток двигателя (1000..3000 мА)


	0xdddd: данные
	------	
	1. ответное слово
